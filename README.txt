Brainfuck

Anton Simmerle
07/02/2024

The Brainfuck repository is a simple Brainf*ck interpreter written in C.

References:
* <https://en.wikipedia.org/wiki/Brainfuck>
* <https://www.youtube.com/watch?v=hdHjjBS4cs8&ab_channel=Fireship>

Usage:

$ gcc -o main main.c
$ ./main input.bf

Input (input.bf):

* Type here your Brainf*ck program.

```
>+++++++++[<++++++++>-]<.
>+++++++[<++++>-]<+.
+++++++.
.
+++.
>>>++++++++[<++++>-]<.
>>>++++++++++[<+++++++++>-]<---.
<<<<.
+++.
------.
--------.
>>+.
>++++++++++.
```

Output (console):

```
Hello World!
```
